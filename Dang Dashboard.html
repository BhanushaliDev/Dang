Methodological note. This dashboard presents the integrated AHP–SCLP–FRM framework applied to 44 GPS-surveyed candidate nodes across Dang District. The Analytic Hierarchy Process (AHP) derives criterion weights; the Set Covering Location Problem (SCLP) minimises warehouse count under service-radius constraints; the Facility Readiness Model (FRM) cross-validates geometric solutions against infrastructure viability. The spatial-infrastructure paradox at Bardipada (FRM = 5.58, below threshold = 6.0) demonstrates the necessity of FRM post-processing for unconstrained SCLP outputs.
